# 退步集——后现代人的生活指南 

- 项目缘起
    - 很多人（包括我）都特别焦虑，焦虑物质欲望、信息过载、自我成长，但又往往自觉或不自觉地以「获取更多」的方式来缓解焦虑，而非主动降低欲求。为什么不主动降低标准？部分原因可能是没有合适的「最小行动」可供选择。「退步集」就是提供某种最小行动方案的资讯平台。同时，再搭配一个「最小故事」，用于生动展现「最小行动」的可能性和价值。
